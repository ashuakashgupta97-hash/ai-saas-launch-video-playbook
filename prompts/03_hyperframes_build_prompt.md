# Prompt 03: HyperFrames Build Prompt

Use this when turning the storyboard into a HyperFrames composition.

```text
Build a HyperFrames video composition from the storyboard.

Requirements:
- 1920x1080
- 90 seconds
- HTML, CSS, and GSAP animation
- use a clean enterprise SaaS visual style
- use product screenshots as the main proof layer
- add focus boxes around important UI areas
- add subtitles in a dark translucent strip near the bottom
- keep text large and readable
- avoid generic AI robot imagery

The motion style should include:
- document cards flowing into an intake engine
- current-process burden steps appearing quickly
- transformation from many manual steps into three review steps
- screenshot zooms and focus rectangles
- KPI cards at the end

Add placeholders for screenshots if actual screenshots are not available.
Keep all content anonymized.
```
