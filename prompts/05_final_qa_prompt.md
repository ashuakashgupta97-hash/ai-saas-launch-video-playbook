# Prompt 05: Final QA Prompt

Use this before publishing the final video.

```text
Please run a final QA pass on the launch video.

Check:
- voiceover starts and ends cleanly
- subtitles match the spoken words
- scene changes happen at the right voiceover moments
- screenshots are readable
- focus rectangles are aligned
- no confidential names, logos, emails, customer names, or internal IDs are visible
- no subtitles overlap important UI
- final KPI screen is readable
- video duration is within 75-90 seconds

Create:
- final MP4
- 4-5 short clips for demos or portfolio use
- a list of exact output paths
- a short release note describing what was verified
```
