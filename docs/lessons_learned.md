# Lessons Learned

## 1. Start With Business Friction

The strongest videos do not start with the technology. They start with a painful workflow people recognize.

## 2. One Good Script Is Not Enough

The script, screen changes, subtitles, and animations must all say the same thing at the same time.

## 3. Screenshots Create Trust

Abstract animations explain the idea. Product screenshots prove the product exists.

## 4. AI-Assisted Review Works Best Iteratively

The strongest results came from multiple review rounds:

- script draft
- storyboard
- visual direction
- animation build
- subtitle sync
- QA review
- final polish

## 5. Timestamp Feedback Matters

Feedback like "the highlight should appear when this exact line is spoken" is much more useful than "make it better."

## 6. Captions Need Human Review

Automatic transcript output is a good starting point, but public video subtitles should be manually checked for wording, wrapping, and timing.

## 7. Public Release Review Is Part Of The Workflow

If the video comes from an internal product, public release review should not be an afterthought. It should be planned before publishing.
